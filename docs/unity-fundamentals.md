# 🎮 Unity Temelleri (Unity Fundamentals)

Unity, dünyanın en popüler oyun motorlarından biridir. Kullanıcı dostu arayüzü, güçlü topluluğu ve çoklu platform desteği sayesinde hem bağımsız geliştiriciler hem de profesyonel stüdyolar tarafından kullanılmaktadır.

---

# 🖥️ Unity Editor

Unity Editor, oyunun geliştirildiği çalışma ortamıdır.

Temel pencereler:

| Pencere | Görevi |
|---------|---------|
| Scene | Oyun dünyasını düzenler |
| Game | Oyuncunun göreceği ekran |
| Hierarchy | Sahnedeki nesneler |
| Inspector | Seçilen nesnenin özellikleri |
| Project | Dosyalar ve Asset'ler |
| Console | Hata ve log kayıtları |

# 💻 İlk Unity Script'i

```csharp
using UnityEngine;

public class HelloUnity : MonoBehaviour
{
    void Start()
    {
        Debug.Log("Hello Unity!");
    }
}
```

Bu kod oyun başladığında Console penceresine mesaj yazdırır.

---

# 🎮 Unity'de Kullanımı

Yeni bir Unity projesinde;

1. Scene oluştur.
2. GameObject ekle.
3. Script oluştur.
4. Script'i nesneye bağla.
5. Play tuşuna bas.
6. Console'u kontrol et.

---

# 📌 Best Practices

✅ Proje klasörlerini düzenli tut.

✅ Asset isimlerini anlamlı ver.

✅ Prefab kullan.

✅ Script dosyalarını kategorilere ayır.

✅ Sık sık Git commit'i yap.

# ⚠️ Yaygın Hatalar

❌ Hierarchy'yi düzensiz bırakmak

❌ Script isimleri ile class isimlerini farklı yapmak

❌ Gereksiz Asset indirmek

❌ Console hatalarını görmezden gelmek

❌ Her şeyi tek Scene'e koymak
