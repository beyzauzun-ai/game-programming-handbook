# 🎮 Game Programming Handbook

A comprehensive Turkish handbook based on the  
**From Code to Creation: Mastering Game Programming** course.

This repository contains structured study notes covering the fundamentals of game development, game programming, Unity, artificial intelligence, animation, optimization, project management, and game production.

> 📚 This project is currently being transformed into a documentation website using **MkDocs Material**.

---

## 📖 About the Project

This handbook was created to organize and reinforce the concepts I learned throughout the course.

Rather than keeping the content as simple course notes, my goal is to build a clear, searchable, and continuously improving Turkish reference guide for game development.

The project will include:

- Detailed topic explanations
- Unity-related examples
- Code snippets
- Quick review sections
- Technical interview questions
- A game development glossary
- Course quiz summaries
